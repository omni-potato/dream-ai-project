# \# English Guardian Prototype

# 

# English Guardian is an English-learning game prototype built with Python and Streamlit.

# 

# Players feed the character English sentences, collect phrasal verbs, develop attributes, and review their learning history.

# 

# \## Current Features

# 

# \- Main screen

# \- Feed screen

# \- Feed history

# \- Six attribute points

# \- Dictionary and rare collection counters

# \- Six-hour feeding cooldown

# 

# \## Run

# 

# Install the required package:

# 

# ```bash

# python -m pip install -r requirements.txt

# ```

# 

# Start the application:

# 

# ```bash

# python -m streamlit run main.py

# ```
